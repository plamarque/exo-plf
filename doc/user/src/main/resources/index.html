Gatein

GateIn portal framework is an open source project developed jointly by Red Hat / JBoss. It provides a powerful framework for developing portlets and other web-based user interface.

eXo Content

eXo Content combines several eXo open source project to provide eXo Platform 3 with comprehensive Enterprise Content Management (ECM) capabilities. Quickly build websites, add document management and workflow to portal-based apps, and much more.
